## Project Overview

OQULIX is focused on creating immersive technology solutions that enhance healthcare, education, and enterprise innovation. The website is designed to reflect the premium nature of the brand, featuring smooth animations, high-quality visuals, and an engaging user experience.

## Technologies Used

- React.js
- Tailwind CSS
- Framer Motion
- GSAP
- Three.js
- Lenis smooth scroll

## Installation

To get started with the project, clone the repository and install the dependencies:

```bash
git clone https://github.com/yourusername/oqulix-website.git
cd oqulix-website
npm install
```

## Usage

To run the development server, use the following command:

```bash
npm start
```

The application will be available at `http://localhost:3000`.

## Components

The website is structured into several key components:

- **Hero Section**: Full-screen cinematic hero with rotating product slides.
- **Product Showcase**: Horizontal scrollable slider for showcasing products.
- **Solutions Section**: Cards representing various solution categories.
- **About OQULIX**: Interactive timeline and animated counters.
- **3D Experience Section**: 3D rotating objects showcasing products.
- **Client Testimonials**: Auto slider for client feedback.
- **Team Section**: Animated profile cards for team members.
- **Contact Section**: Modern contact form with Google Maps integration.
